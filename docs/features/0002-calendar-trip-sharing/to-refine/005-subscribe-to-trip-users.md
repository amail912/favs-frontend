# Subscribe To Trip Users

## Goal
As a user, I want to choose whose trips I follow so my calendar only surfaces the travel information that matters to me.

## How This Story Achieves The Goal
The application should let each user maintain the list of people they subscribe to. This is the second user-managed list involved in shared trip visibility.

## Technical Details
Provide a manageable UI for viewing, adding, and removing users from the subscription list. The UI should make it clear that subscribing alone is not enough for visibility if the other user does not share their trips back.

## Tests
- Integration tests for rendering and updating the subscription list UI.
- E2E tests covering adding and removing a user from the subscription list.
